**Description of Sections Implemented**

This project is a fully responsive Fintech Landing Page built using semantic HTML5 and CSS3.
The website includes the following sections:

_**Header + Navigation:**_ Logo, navigation links, skip-to-content link, call-to-action button.

_**Hero Section:**_ Large headline, sub-text, CTA buttons, hero image with fade-in animation.

_**Trusted Brands:**_ Logo strip with hover effects.

_**Features Section:**_ Grid of feature cards using CSS Grid and custom icons.

_**How It Works section:**_ Three-step process layout with animated step bubbles.

_**Pricing Section:**_ Three pricing cards (basic, standard, pro).

_**Testimonials:**_ Grid of user feedback cards.

_**Newsletter:**_ Email subscription form.

_**Contact Form:**_ Two-column form (collapses on mobile).

_**Footer:**_ Five-column footer with links and copyright.

_**Back-to-Top Button:**_ Sticky button with hover animation.


**2. Breakpoints Used & Layout Techniques**

**Breakpoints**

_**1200px: Large-tablet adjustments**_ (reduced spacing, resized hero image, responsive grids).

_**992px: Tablet layout**_ (hero becomes column, grids switch to 2-column, footer adjusts).

_**768px: Mobile layout**_ (nav becomes toggle icon, all grids become 1 column).

_**480px: Small-mobile layout**_ (font-size adjustment, newsletter stacked, full-width buttons).

**Layout Techniques**

_**CSS Flexbox:**_ Header, hero layout, buttons, navigation alignment, forms.

_**CSS Grid:**_ Features, pricing, testimonial cards, footer layout, steps grid.

_**CSS Variables:**_ Colors, spacing scale, border radius, typography.

_**Animations & Hover Effects:**_ Keyframe fade-in, lift-on-hover cards, logo hover grayscale, button transitions.
