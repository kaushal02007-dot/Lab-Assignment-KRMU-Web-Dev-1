**JavaScript Quiz Game**
---This project is a simple JavaScript quiz game created for Web Development I, Lab Assignment 4.
---The quiz runs completely in the browser console using prompt() and alert().

**How the Quiz Works**
--All quiz questions and answers are stored inside an array called quizQuestions.
--A function named runQuiz() controls the entire quiz process.
--Each question is shown using prompt().
--The userâ€™s answer is converted to lowercase and trimmed to remove extra spaces.
--The answer is checked using an if statement.
--Feedback is shown after every question.
--At the end, the program displays the final score.

**Features Used**
1. Arrays
2. Functions
3. For loops
4. Conditional statements
5. prompt() and alert()
6. String methods: toLowerCase() and trim()

**How to Run the Quiz**
1. Open a web browser.
2. Open the Developer Tools console (F12 or Ctrl + Shift + I).
3. Go to the Console tab.
4. Paste the JavaScript quiz code.
5. Press Enter to run the quiz.
